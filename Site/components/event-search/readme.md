# Profile-Card

Diese WebComponente zeigt die Daten eines Benutzers an und macht Änderungen daran möglich.

Eingabeparameter:
  - username (Benutzername)
  - birthdate (YYYY-MM-DD Format)
  - mySports (Auflistung von Sportarten)
  - about (Über mich)
