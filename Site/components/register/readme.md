# Register-Component

Diese WebComponente gibt das Formular zum Registrieren aus. Dabei werden folgende Registrierungsdaten abgefragt:
  - Name
  - Vorname
  - Email-Adresse
  - Geburtstag
  - Benutzername
  - Region
  - Veranstaltungsart
  - Passwort
  - Passwort wiederholen
  - AGB akzeptieren Checkbox
  - Registrieren-Button
